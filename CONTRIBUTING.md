# CONTRIBUTING.md

## How to Contribute
- Fork the repo, create a branch, submit a PR.
- Write clear commit messages.
- Add/maintain tests for new features or bugfixes.
- Update docs as needed.

## Code Style
- Use clear, concise code and comments.
- Prefer type hints and docstrings.
- Follow PEP8.

## Reporting Issues
- Use GitHub Issues.
- Provide steps to reproduce, expected/actual behavior, and environment info.
